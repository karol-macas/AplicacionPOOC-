INSTRUCCIONES DE EJECUCION
-Para ejecutar el programa debes tener instalado SDK.NET 

1. Abrir el proyecto en Visual Studio Code 
    Abrir el terminal y si quieres buscar el directorio del archivo
    del proyecto en C#. Puedes buscar como:
    -cd ruta de tu archivo

2. Compilacion y Ejecutar el programa
    Para ejecutar y compilar el programa se utiliza el comando dotnet run 
    -dotnet run 
    
3. Resultados 
    Una vez ingresado el comando de ejecutar  observaras la Informacion del Empleado y Gerente en la consola
